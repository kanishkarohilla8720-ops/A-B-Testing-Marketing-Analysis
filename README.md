# A/B Testing Analysis — Marketing Campaign Performance

## Overview
This project analyzes the performance of two marketing campaign variants (A and B) to determine which version drives higher conversions.

## Tools Used
- Excel
- Python (Pandas, NumPy)
- Matplotlib / Seaborn

## Key Steps
- Cleaned and prepared dataset
- Calculated conversion rate, CTR, uplift %
- Conducted statistical significance testing
- Visualized results

## Insights
- Variant B outperformed Variant A
- Conversion uplift observed
- Recommended scaling Variant B

## Skills Demonstrated
A/B Testing, Statistical Analysis, Data Cleaning, Visualization
